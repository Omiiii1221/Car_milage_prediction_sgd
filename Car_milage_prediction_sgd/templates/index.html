Project Details: Vehicle Mileage Prediction

This project focuses on leveraging Regression Analysis to predict vehicle performance, specifically measuring mileage in kilometers traveled per liter. Our core objective is to build a robust and accurate predictive model using the Stochastic Gradient Descent (SGD) algorithm.

By analyzing various vehicle attributes, the model aims to accurately estimate fuel efficiency, providing valuable data-driven insights. This helps both consumers in making informed purchasing decisions and manufacturers in optimizing vehicle design for better fuel economy.
